Hikama 3.0
============
---Popis---
Hikama 3.0 je program pre testovanie znalostí o derivátoch máp, ktoré 
boli používané na našom území.
Pred použitím odporúčame prečítať aj súbor "Pouzitie.pdf", kde je
bližšie vysvetlené používanie súboru aj s grafickou pomocou.


---Príklad použitia---
Súbor spustíte pomocou Hikama3.exe alebo pomocou python kódu Hikama3.py


---Licencia---
Súbor je voľne šíriteľný


---Ďalšie informácie---
Program bol vytvorený pomocou python kódu a následne exportovaný cez
pyinstaller. Z tohto dôvodu je možné očakávať problémy s antivírusom 
pri spustení. V takomto prípade odporúčame spúšťať Hikama3.py podľa
pokynov v súboru "Pouzitie.pdf".
